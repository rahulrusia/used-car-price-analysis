# 🚗 Used Car Price Analysis

## 📌 Overview
This project analyzes what factors influence used car prices using machine learning.

## 🎯 Objective
Identify key drivers of price and build predictive models to support dealership decisions.

## 📊 Dataset
- Source: Kaggle Used Cars Dataset
- Size: ~426K records

## 🔍 Key Insights
- Newer cars → higher prices
- Lower mileage → higher value
- Condition significantly impacts price
- Brand plays a major role

## 🧠 Models Used
- Linear Regression
- Ridge / Lasso
- Random Forest
- SVM


## 💼 Business Recommendations
- Focus on newer, low-mileage vehicles
- Invest in improving vehicle condition
- Prioritize high-value brands
